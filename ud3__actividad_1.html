UF2215. UD3. Actividad 1. Ejercicio teórico-práctico: Funcionalidades CASE


Preguntas / Actividades a realizar 

Pregunta 1. Realiza un cuadro comparativo donde se clasifique las diferentes herramientas CASE indicando sus características y sus funcionalidades principales.

UD3 - Actividad 1 - Resolución

Pulsa aquí para visualizar: UD3 - Actividad 1 - Resolución

Herramienta Características principales Funcionalidades destacadas
IBM Rational Rose - Amplia adopción en empresas y proyectos de gran envergadura.
- Integración con otras herramientas de desarrollo IBM.
- Soporte para la creación de diagramas de clase, secuencia, actividad, etc.		 - Modelado UML para el análisis y diseño de sistemas.
- Generación de código en varios lenguajes de programación.
- Herramientas avanzadas para la colaboración en equipo.
Microsoft Visio - Popular entre usuarios de Microsoft Office.
- Interfaz intuitiva y fácil de usar.
- Soporte para la creación de diagramas UML.		 - Amplio conjunto de plantillas para diagramas de diversos tipos.
- Integración con otras aplicaciones de Microsoft Office.
- Colaboración en tiempo real a través de Microsoft 365.
Oracle SQL Developer - Específico para el desarrollo y administración de bases de datos Oracle.
- Integración con Oracle Database y Oracle Cloud.
- Soporte para la gestión de cambios en esquemas de bases de datos.		 - Modelado de bases de datos y generación de scripts SQL.
- Herramientas para la ejecución y optimización de consultas SQL.
- Visualización y análisis de datos a través de consultas SQL.
Sparx Enterprise Architect - Herramienta completa que cubre todo el ciclo de vida del software.
- Amplias capacidades para la gestión de requisitos y trazabilidad.
- Colaboración en equipo y gestión de cambios.		 - Modelado UML y soporte para otras notaciones y estándares.
- Generación de código en varios lenguajes y frameworks.
- Integración con herramientas de desarrollo y repositorios de código.
Visual Paradigm for UML - Enfoque específico en modelado UML.
- Herramientas para la generación de código y reversión de ingeniería.
- Colaboración en equipo y gestión de versiones.		 - Amplia variedad de diagramas UML y otras notaciones.
- Simulación de procesos y generación de documentación detallada.
- Integración con sistemas de control de versiones como Git.
ArgoUML - Herramienta de modelado UML de código abierto y gratuita.
- Plataforma independiente y extensible.
- Comunidad activa de usuarios y desarrollo.		 - Soporte para diversos tipos de diagramas UML.
- Generación de código en lenguajes como Java y C++.
- Herramientas para la revisión y anotación de modelos.
Poseidon - Enfoque en modelado UML y desarrollo ágil.
- Integración con sistemas de control de versiones.
- Herramientas para la simulación de procesos y pruebas.		 - Soporte para diagramas UML y notaciones específicas de desarrollo ágil.
- Generación de código en diversos lenguajes.
- Colaboración en equipo a través de la nube.
ERwin - Enfoque específico en modelado de bases de datos.
- Amplia compatibilidad con diferentes sistemas de gestión de bases de datos.
- Herramientas para la gestión de metadatos y versiones.
- Integración con otras herramientas CASE y de desarrollo.		 - Modelado visual de esquemas de bases de datos.
- Generación automática de scripts SQL.
- Colaboración en equipo para el desarrollo y mantenimiento de bases de datos.
- Soporte para la documentación detallada de esquemas y relaciones.
EasyCASE - Enfoque integral que abarca diversos aspectos del desarrollo de software.
- Herramientas para el análisis de riesgos y la gestión de proyectos.
- Interfaz gráfica intuitiva y fácil de usar.		 - Modelado de sistemas, bases de datos y flujos de trabajo.
- Generación de código en varios lenguajes de programación.
- Soporte para la documentación y seguimiento de cambios en los modelos.
Oracle Designer - Específico para el desarrollo en entornos Oracle.
- Integración con el resto de herramientas Oracle.
- Soporte para la definición de reglas de negocio.		 - Modelado de bases de datos y flujos de trabajo.
- Generación de scripts y código PL/SQL (Procedural Language/Structured Query Language).
- Herramientas para la gestión de cambios y la colaboración en equipo.
PowerDesigner (SAP) - Enfoque integral en el ciclo de vida del software.
- Integración con sistemas SAP y otras plataformas.
- Soporte UML y generación de código en varios lenguajes.		 - Modelado de sistemas, datos y procesos de negocio.
- Análisis de impacto y gestión de dependencias entre componentes.
- Herramientas avanzadas para la documentación y la visualización de modelos.
SNAP - Herramienta de modelado y análisis de sistemas.
- Enfoque en la simplicidad y facilidad de uso.
- Generación de documentación detallada.
- Soporte para la creación de prototipos rápidos.		 - Modelado visual de sistemas y flujos de datos.
- Análisis de impacto y gestión de dependencias.
- Colaboración en tiempo real y seguimiento de cambios.
- Integración con el desarrollo ágil y otras metodologías de desarrollo de software.
