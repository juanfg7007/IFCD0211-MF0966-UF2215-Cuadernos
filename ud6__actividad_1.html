Preguntas / Actividades a realizar 

Pregunta 1. Realiza un cuadro comparativo donde se muestren las características generales de pasarelas y medios de conexión para permitir su selección y utilización en los procedimientos de extracción y consulta de información.

Características Pasarelas Medios de conexión
Definición Son interfaces que permiten la comunicación entre sistemas heterogéneos, facilitando el acceso y manipulación de datos entre diferentes bases de datos. Son mecanismos que posibilitan la conexión entre aplicaciones y bases de datos, gestionando la transferencia de datos y consultas.
Funcionalidad Transforman y traducen las consultas realizadas desde una aplicación a un formato comprensible para la base de datos subyacente. Establecen y mantienen la comunicación entre la aplicación y la base de datos, gestionando la transferencia eficiente de datos.
Compatibilidad de plataformas Específicas para ciertas combinaciones de sistemas gestores de bases de datos, actuando como puente entre plataformas diferentes. Suelen ser independientes de la plataforma, ofreciendo flexibilidad para conectarse con diversos sistemas gestores de bases de datos.
Nivel de abstracción Proporcionan un alto nivel de abstracción, ocultando las complejidades técnicas y diferencias de sintaxis entre las bases de datos conectadas. Pueden ofrecer diferentes niveles de abstracción, desde opciones simples para operaciones básicas hasta control más detallado para consultas especializadas.
Configuración y mantenimiento Requieren configuración específica para cada combinación de sistemas, lo que puede resultar en un mayor esfuerzo de mantenimiento. Al ser más genéricos, suelen ser más fáciles de configurar y mantener, especialmente en entornos donde se utilizan diversas bases de datos.
Rendimiento Puede haber una ligera degradación del rendimiento debido a la traducción y adaptación de consultas entre sistemas diferentes. El rendimiento puede ser óptimo, ya que están diseñados para gestionar eficientemente la comunicación entre la aplicación y la base de datos.
Escalabilidad La escalabilidad puede depender de la compatibilidad entre las bases de datos conectadas y las limitaciones de la pasarela. Suelen ser escalables, adaptándose a entornos con un crecimiento en la demanda de acceso y manipulación de datos.
