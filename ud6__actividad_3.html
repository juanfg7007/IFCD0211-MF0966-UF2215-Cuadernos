UF2215. UD6. Actividad 3. Caso práctico: Parámetros ODBC


Preguntas / Actividades a realizar 

Pregunta 1. Configurar todos los parámetros ODBC de MySQL en Windows 7. Explica los pasos a seguir para realizar correctamente esta configuración ODBC.

UD6 - Actividad 3 - Resolución

Pulsa aquí para visualizar: UD6 - Actividad 3 - Resolución

Pasos para configurar correctamente un ODBC para MySQL en Windows 11 (pese a que el enunciado se indica Windows 7, he preferido describir las instrucciones con el sistema operativo actual):

  • Descargar el controlador ODBC de MySQL
    Acceder al sitio web oficial de MySQL o de Oracle para descargar el controlador ODBC adecuado para el sistema operativo Windows 11. 

  • Instalar el controlador ODBC
    Una vez descargado el controlador, ejecutar el archivo de instalación según las instrucciones del asistente de instalación para completar el proceso. Seleccionar la opción para instalar el controlador ODBC durante el proceso de instalación.

  • Abrir el Administrador de orígenes de datos ODBC
    Buscar y abrir el Administrador de orígenes de datos ODBC desde el menú de inicio o utilizando la barra de búsqueda de Windows.

  • Agregar un nuevo origen de datos
    En el Administrador de orígenes de datos ODBC, ir a la pestaña DSN de usuario. Hacer clic en el botón "Agregar" para iniciar el proceso de configuración del nuevo origen de datos.

  • Seleccionar el controlador MySQL ODBC
    En la ventana de configuración, elegir el controlador MySQL ODBC instalado previamente de la lista de controladores disponibles.

  • Configurar los parámetros de conexión
    Completar los campos requeridos para configurar la conexión a la base de datos MySQL. Esto incluye proporcionar la dirección del servidor MySQL, el nombre de la base de datos, el nombre de usuario y la contraseña. Especificar otros parámetros según sea necesario, como el puerto de conexión y las opciones de seguridad.

  • Probar la conexión
    Una vez configurados todos los parámetros, hacer clic en el botón "Probar conexión" para verificar si la configuración es correcta y establecer una conexión exitosa con la base de datos MySQL.

  • Guardar la configuración
    Si la prueba de conexión es exitosa, hacer clic en "Aceptar" o "Guardar" para guardar la configuración del nuevo origen de datos ODBC. Esto agregará el nuevo origen de datos a la lista de orígenes de datos disponibles en el Administrador de orígenes de datos ODBC.

  • Cerrar el Administrador de orígenes de datos ODBC
    Una vez completada la configuración, cerrar el Administrador de orígenes de datos ODBC para finalizar el proceso.
