UF2215. UD5. Actividad 1. Caso práctico: PL / SQL


Enunciado (Descripción)

Un precompilador es una herramienta de programación que permite incluir sentencias de SQL o de PL/SQL en un programa escrito en un lenguaje de alto nivel. Oracle tiene precompiladores para Ada, C,C++,COBOL, FORTRAN, Pascal y PL/I.

El precompilador acepta como entrada un programa fuente y traduce las sentencias de SQL embebidas en él, en llamadas a rutinas de Oracle, generando un programa que se compila, enlaza y ejecuta de forma habitual.

Vamos a ver como se realizaría un programa PL/SQL en Oracle y para ello vamos a disponer de la siguiente tabla:

CREATE TABLE Clientes(
Id_ClienteVARCHAR(3) PRIMARY KEY,
Descripcion_ClienteVARCHAR(40) NOT NULL);

Y de los siguientes valores:

Insert INTO Clientes (Id_Cliente,Descripcion_Cliente) VALUES ('100','Juan Martos Galán');
Insert INTO Clientes (Id_Cliente,Descripcion_Cliente) VALUES ('150','La papelería,s.a');
Insert INTO Clientes (Id_Cliente,Descripcion_Cliente) VALUES ('160','Librería Técnica s.l');
Insert INTO Clientes (Id_Cliente,Descripcion_Cliente) VALUES ('200','María López Pérez');
Insert INTO Clientes (Id_Cliente,Descripcion_Cliente) VALUES ('210','José Martín Rubio');


Preguntas / Actividades a realizar 

Pregunta 1. Hacer un programa PL/SQL que lea el registro correspondiente al cliente de identificador 200 y almacene su contenido en una variable de tipo registro.

Pregunta 2. Hacer un programa PL/SQL que almacene en una tabla llamada vi_matriz_cliente en el índice 160 el registro correspondiente al cliente 160.
